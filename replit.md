# مدیریت قرض - Loan Manager App

A Persian-language loan management mobile app built with Expo React Native.

## Features
- Weekly candle-style bar chart showing daily loan totals
- Add persons with name and phone number
- List all persons with their total debts
- Add loan entries with amount, Persian date (auto-filled), and notes
- Reports: daily, weekly, monthly, and overall
- PDF export with sharing

## Tech Stack
- Expo Router (file-based routing)
- AsyncStorage for persistence
- React Context for state
- expo-print + expo-sharing for PDF export
- Persian date conversion utility
